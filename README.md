# Quiz Wiz

Quiz-Wiz is a web-based quiz application built using Java and Thymeleaf for server-side rendering. 
The project is aimed to provide a simple yet effective quiz interface for users. 
For educators, trivia enthusiasts, or anyone looking to assess skills in a variety of subjects.

### Technologies Used

   * Java 17
   * Spring Boot 3.1.4
   * Thymeleaf
   * Hibernate
   * SQLite
   * Maven
   * MapStruct

### Setup

To run this project, make sure you have Java and Maven installed.
